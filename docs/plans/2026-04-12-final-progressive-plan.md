# The Real Plan — Progressive Build from 9km to Sub-17:30 / Sub-36

**Date:** 2026-04-12
**Starting point:** 9km longest run post-marathon, ~22km/week, detrained
**Goals:** Sub-17:30 5K, Sub-36 10K. No fixed dates — race when ready.

## Guiding Principles

1. **Earn each phase.** No jumping ahead. Long run builds 2km/week max.
2. **Hard days hard, easy days easy.** Gym on Tue PM (after track). Yoga Fri.
3. **Thursday fartlek phases in gradually.** First 4 weeks = easy run. Then introduce.
4. **Sunday is always the long ride.** Non-negotiable.
5. **Bike carries intensity when running can't.** Sweet spot / tempo bike on Monday when Thursday is easy.
6. **Bailout rules everywhere.** HR accountability. If pace drifts 3sec, stop.

## What's different from every previous version

1. **10km → 24km long run build over 11 weeks** (not jumping to 20km week 1)
2. **No fartlek weeks 1-4** — Thursday is easy running only
3. **Bike intensity fills the gap** — Monday sweet spot when Thursday is easy
4. **Sunday long ride kept** — gym moves to Tue PM
5. **Yoga on Friday** — dedicated slot
6. **No fixed race date** — race when the assessment says ready (~Jul 26 for 5K, ~Aug 23 for 10K)
7. **Proper recovery weeks** — 26-37km, 12-16km long run, reduced quality
8. **Assessment checkpoint at Week 12** — 3×1km at 3:28-3:30 decides if we race or keep building
