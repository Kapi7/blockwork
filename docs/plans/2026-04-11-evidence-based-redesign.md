## The Foundation Changes

### 1. New weekly template — Norwegian Singles + Canova specific

**Ditch the Tue/Thu double-KEY model.** It violates 72-hour CNS recovery on a 4-run week.

```
Mon: Easy bike 60-90min Z2 (aerobic base, no impact)
Tue: KEY — Sub-threshold intervals (primary quality)
Wed: Easy run 6km + 4×100m strides (run frequency, no CNS cost)
Thu: Gym + easy bike recovery 45min (strength day)
Fri: Easy run 5km (shake) OR rest
Sat: Long run with specific insert (secondary quality — Canova-style)
Sun: Bike Z2 long 90-120min (aerobic volume) OR rest
```

**4 runs (Tue/Wed/Fri/Sat), 3 bikes (Mon/Thu/Sun), 1 gym (Thu).**

**Why this works:**
- Tue KEY → Sat long run = 96 hours of pure easy recovery. CNS fully replenished.
- Gym on Thu doesn't interfere with any KEY run.
- Wed easy + strides maintains neuromuscular patterns without fatigue.
- Fri shake run is optional — skip it if the body's talking back.

### 2. Sub-threshold replaces threshold as primary quality

**The data says:** In 18 months, zero runs averaged faster than 3:49/km. You've been racing at threshold, but you've never TRAINED at threshold volume. You can't build something you won't rehearse.

**The fix:** Norwegian Singles sub-threshold. Sessions target **3:50-3:58/km** (5-15 sec slower than the 3:45 LTP). You finish EVERY rep feeling you could do 2 more. No burn, no blow-up.

**New primary session structures (Tue KEYs):**
- `subThreshold4x2km` — 4×2km @ 3:50-3:55/km, 90s jog (≈95-97% threshold)
- `subThreshold6x1km` — 6×1km @ 3:48-3:52/km, 60s jog (Bakken format)
- `subThreshold3x3km` — 3×3km @ 3:52-3:55/km, 2min jog (later in block, bigger reps)
- `subThreshold45_15` — 10×(45s @ 3:35/km / 15s jog), 3min rest, 3 sets (neuromuscular + lactate)

### 3. 5K pace is introduced GRADUALLY — you've never done it

**Critical gap from data:** Fastest 1km ever in 18 months = 3:34/km (one rep). Sub-17:30 requires 3:29/km. Jumping straight to 5×1km at race pace in week 5 = injury + failure.

**On-ramp:**
- **Weeks 1-4 (Block 1):** No 5K pace yet. Build sub-threshold base. Strides at feel (3:35-3:45 for 100m is fine, no sustained reps).
- **Week 5:** Introduce 5K pace with **very short reps**: 8-10×200m @ 3:25-3:30/km (~40sec) with full 200m jog recovery. Short enough to be neuromuscular, not aerobic.
- **Week 6:** 8-10×400m @ 3:28-3:32/km (~82sec) with 200m jog.
- **Week 7:** 6-8×600m @ 3:28-3:32/km with 200m jog.
- **Week 8:** Recovery week.
- **Week 9 (Block 3):** 5×1km @ 3:28-3:32/km — first full-length reps at goal pace.
- **Week 10:** 6×800m @ 3:25-3:28/km (slightly faster than goal).
- **Week 11:** 3-4×1km @ 3:25-3:30/km (sharpener, low volume).
- **Week 12:** Race week + 5K TT.

### 4. Block 1 18km test → capped at Z3 (no Z5a surge)

**Problem:** Current test jumps from 7km recovery runs to 18km with the last km at HR 175+ (Z5a). This is how the marathon blew up.

**Fix:** 18km test becomes a pure **aerobic discipline test**. Cap at HR 165 (top of Z3 Tempo). If HR creeps above → WALK until it drops. The test is not "go fast" — the test is "stay in zone when tired." This directly rehearses the lesson from Limassol.

### 5. Block 4 gets a 3rd 10K-pace session + protected recovery

**Problem 1:** Only 2 sessions at 10K pace before the 10K TT. Too thin for a 51-second PB drop.
**Problem 2:** First hard session at T+5 days post 5K TT. Not enough CNS recovery.

**Fix:**
- **Week 13 (Jul 6-12):** Genuine recovery, no hard runs. Easy runs + bike.
- **Week 14 (Jul 13-19):** First 10K-pace session = Tue 4×2km @ 3:38-3:40/km. Long run Sat 16km with last 6km @ 10K pace.
- **Week 15 (Jul 20-25):** Tue second 10K-pace session = 3×3km @ 3:35-3:38 (continuous, the big specific test). Long run 12km with 4km @ 10K pace. 
- **Week 16 (Jul 20-26):** Taper + 10K TT. Wait — that's overlap.

Reality check: the Jul 25 10K TT is 20 days post Jul 5 5K TT. That's tight. Let me keep it at 3 weeks:
- **Week 13 (Jul 6-12):** Recovery
- **Week 14 (Jul 13-19):** 10K specific — 2 quality sessions at 10K pace
- **Week 15 (Jul 20-25):** Taper + TT

Still thin. Consider moving 10K TT to **Aug 1** (Sat) — that gives 4 weeks of 10K prep. Worth asking the user.

### 6. Strength work upgraded — heavy + plyometric

**Evidence:** Strength training improves running economy 3-8% (meta-analyzed). Your single biggest ROI lever.

**New gym protocol (Thu):**
- **Blocks 1-2 (strength build):** Heavy squats 3-5×5 @ 80-85% 1RM, single-leg RDL 3×8, box jumps 3×5, bounding 3×20m, calf raises 3×15.
- **Block 3 (power):** Front squats 3×3 @ 85-90%, single-leg hops 3×8, depth jumps 3×5, reactive strength work. Drop volume, keep intensity.
- **Block 4 (maintain):** 2×5 back squats, bounding, core. Light stimulus, no DOMS.

### 7. HR accountability loop (fix the marathon lesson)

**Add a check to George:**
- Weekly: Tue KEY avg HR vs planned. If >5 bpm above target → Fri session becomes easy.
- Monthly: On the bike, 10min @ "comfortably hard" — aerobic decoupling <5% = base is working.
- Race week: T-4 stride opener. If HR on 200m strides >85% max at target pace → flag fatigue.

This isn't a "nice to have." The marathon blew up because HR was ignored. Without an enforcement loop, it happens again.
